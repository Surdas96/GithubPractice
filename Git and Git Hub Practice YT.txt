-Create one file under created folder in vs code (name index.html and type code any eg. select html:5 )
-Run git init

-Run git commit -m "First version index.html"
-To check branch run git branch
-git log (Show all info)
-Now make some changes in code and save
-And again run git status you will see modified index.html 
-Run git diff  (Shows what changes you have made)
-Run git add .
-Run git status
-Now commit again
-Run git commit -m "Second commit index.html"
-Same make change and do Third commit

#To restore file before git add . command
-git restore . (Restore all the last changes made and shows previous version)

#To restore file after doing git add . command
-git restore --staged . 
 your file return back from staging area
-Run git restore . 

#To restore filer after git add. and again made new changes before commit
-Run git restore --worktree . 
your last mistake will be remove 

#To restore file after commit 
-Run git reset --soft Head^
-Run git reset --hard Head^  (HEAD^ for previous version)

* Soft reset: it will revert back to last changes and allow you to make changes
* Hard reset: directly came back to last version 

#Different Git Logs Options
-Run git log -p -1 (It shows last commit)
-Run git log -p -2 (Shows last 2 commit )
-Run git log --stat (Summery of changes)
-Run git log --pretty=oneline (Shows commit in one line)
-Run git log --pretty=format:"%h - %an, %ar:%s"
-Run git log -S function_name 
-Run git log --grep="V2" (shows specific msg add msg in bracket )
-Run log --since="date"
-Run log --until="date"
-Run git log --author="name"
-Run git log --no-merges

# Git Branching and Merging

-Run git branch

-Create new file or change any changes in codes. 
- Now commit
-Run git commit -m "New branch new file"

# To merge design branch to main
-Go to 1st main branch -git checkout main
